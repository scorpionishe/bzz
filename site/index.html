Как работает

Три шага. Меньше секунды.

01

Перехватывает ввод

Мониторит нажатия клавиш через Accessibility API macOS. Работает в фоне, не нагружая систему.

02

Анализирует слово

Как только ты заканчиваешь слово, Bzz проверяет его по словарю с морфологическим анализом.

03

Исправляет мгновенно

Если слово распознано как транслит, приложение заменяет текст и переключает раскладку одним действием.

FAQ

Часто задаваемые вопросы

Как Bzz отличает намеренный английский текст от транслитерации? +
Приложение проверяет слово по словарю. Если слово, набранное латинскими буквами, не является английским словом (например, «ghbdtn»), но после перекодировки с EN-раскладки на RU получается осмысленное русское слово — Bzz делает исправление. Английские слова типа «hello» остаются нетронутыми.
Нужны ли права администратора для установки? +
Для работы с Accessibility API macOS потребуется выдать разрешение в Системных настройках → Конфиденциальность и безопасность → Универсальный доступ. Это стандартное требование для всех приложений, которые мониторят ввод — аналогично TextExpander, Alfred и другим.
Работает ли Bzz в браузере, терминале, IDE? +
Да. Приложение работает через Accessibility API macOS и функционирует в любом приложении, которое принимает ввод с клавиатуры: Safari, Chrome, VS Code, iTerm2, Slack, Telegram, Pages и т.д.
Замедляет ли приложение ввод? +
Нет. Анализ происходит после окончания слова (пробел или знак препинания) и занимает микросекунды — словарный поиск с хешем. Потребление CPU в фоне составляет <0.1%.
Поддерживаются ли только русский и английский? +
В текущей версии — русский и английский (QWERTY ↔ ЙЦУКЕН). Поддержка дополнительных языков и раскладок планируется в следующих версиях.
Где исходный код? +
Ядро движка (словарь, морфология, алгоритм распознавания) открыто под лицензией MIT на GitHub. Лицензия на платную версию включает GUI-обёртку и интеграцию с macOS.
Что входит в «lifetime license»? +
