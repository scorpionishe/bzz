# Bzz Pro — single source of truth

Этот файл — единственный авторитетный источник Pro-фич Bzz. Все другие материалы
(README, лендинг, бот, маркетинг-черновики) **должны соответствовать этому списку**.

## Free (навсегда, MIT)

Базовый функционал, который **никогда не уйдёт в Pro**:

- Автоисправление слов в неправильной раскладке (98K русский словарь + Snowball stemmer)
- Fuzzy matching опечаток (Levenshtein edit distance 1)
- Перехват Enter до отправки в Slack/Telegram/Notion
- Cmd+Z откат с автообучением исключений per-app
- **Cmd+Shift+X — ручная конвертация выделенного текста** (был ошибочно записан в Pro — это free навсегда, killer feature)
- Tray icon с pause/resume (⚡ / 💤)
- Автозапуск через LaunchAgent
- Per-app исключения через `~/Library/Application Support/Bzz/config.yaml`
- Полностью офлайн, без телеметрии

## Pro — 490₽ разово (v0.4, планируется Q3 2026)

Три фичи, которые добавляются в Pro:

### 1. Custom Dictionary — свой словарь
Пользователь добавляет свои термины, имена клиентов, профессиональный жаргон,
бренды, которые Bzz будет понимать.

**Use case (копирайтер):** "Bzz, перестань исправлять «кэшбэк» на «кексбэк»"
**Use case (журналист):** имена иностранных интервьюеров латиницей в кириллическом тексте

Файл: `~/Library/Application Support/Bzz/custom_dict.txt`
UI: tray menu → "Открыть свой словарь" (открывает в TextEdit)

### 2. Дополнительные раскладки — UK / KZ / BY / DE / FR
Пользователь выбирает дополнительную пару раскладок (одна включена в Pro,
остальные апгрейд за +200₽).

**Use case:** преподаватель украинского в Тбилиси, билингв RU/EN/UA
**Use case:** студент немецкого, RU+DE переписка

### 3. Per-app Exception UI
GUI для управления исключениями (то, что Cmd+Z запоминает автоматически).
Сейчас управление через CLI (`bzz --list-exceptions`, `bzz --forget`).
Pro версия даёт окно: список исключений, удалить, добавить вручную.

**Use case:** программист накопил 100 исключений за месяц, хочет почистить
**Use case:** копирайтер хочет добавить «не трогать слова на латинице в Notion»

## Что в Pro НЕ войдёт (или войдёт позже)

- ❌ Cmd+Shift+X — это **free** и таким останется (см. выше)
- ❌ Whitelist приложений — это уже free через `config.yaml: excluded_apps`
- ❌ Базовый fuzzy — это free
- ⏳ iCloud sync — может быть в v0.5+, требует Apple Developer Program account настройки CloudKit
- ⏳ Темы/кастомизация tray — может быть в v0.5+, low priority
- ⏳ Статистика "сэкономлено X минут" — может быть в v0.5+, конфликтует с zero-telemetry если делать без локального счётчика

## Цена и flow

- **Pro:** 490₽ разовый платёж, lifetime лицензия на 1 Mac
- **Pro+ (раздать другу):** 2 лицензии за 850₽ (-13%)
- **Payment flow:** Telegram-бот `@BzzProBot` (TBA после v0.4) → СБП на номер телефона → Roman подтверждает в admin chat → бот генерирует Ed25519-подписанный JSON → отправляет файл `bzz-license.json`
- **Activation:** в tray menu Bzz → "Активировать Pro лицензию" → выбрать файл из Finder → автоматическая проверка подписи

- ❌ "Может включить" (`may include`) — лучше "Roadmap v0.4: ..."
