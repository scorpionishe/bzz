# Я написал open source альтернативу Caramba Switcher и продаю её разово за 490 ₽

## TL;DR

- На macOS из коробки нет автопереключателя раскладки. Punto Switcher умер в 2017
- Coca есть только Caramba Switcher — подписка 449 ₽/год, закрытый код, у разработчика 70% оценок 1★ в App Store
- Я написал свою — на Go, под MIT, разовая покупка 490 ₽
- Запуск через open source backlinks, SEO под пустые ключи "punto switcher для mac", Telegram-каналы про Mac
- Целевая выручка к году: 200-500 тысяч ₽ при 10K-30K пользователей

## Почему я взялся

Я пишу попеременно на русском и английском по 8-10 часов в день — код-комменты, чаты, документация. Каждые 5 минут забываю переключить раскладку и набираю "ghbdtn" вместо "привет".

На Windows проблему с 2001 года решал Punto Switcher от Yandex. Когда я пересел на Mac в 2019, Punto там был, но даже тогда работал не очень. Сейчас, в 2026, на Apple Silicon он вообще не запускается — последний релиз сентябрь 2017.

Поискал альтернативы:

- **Caramba Switcher** — единственный коммерческий вариант. 449 ₽/год подписка. В App Store рейтинг 4.2 из 5, но из 2 033 оценок 1 408 — единица. Главные жалобы из отзывов: "за что подписка на утилиту", "разработчик грубо отвечает", "невозможно отключить часть фич".
- **Mahou, Kawa, InputSourcePro** — переключают input source по приложению, но не правят уже набранное.

Coca, занимаемое место, отзывы — всё кричало о незанятой нише.

## Что сделал

Утилита называется Bzz. Звукоподражание — звук переключения. Это не "русский переключатель", архитектура поддерживает любые пары раскладок — украинскую, казахскую, белорусскую, немецкую (на roadmap).

**Что делает:**

- Видит, что вы набрали "ghbdtn" — мусор на английской раскладке, осмысленное "привет" на русской — и заменяет автоматически
- Перехватывает Enter в Slack/Telegram/Notion ДО отправки сообщения, исправляет, потом отправляет (Caramba это умеет, Punto не умел)
- Fuzzy matching: даже если вы пропустили клавишу — "gjljk;bv" вместо "ghjljk;bv" — найдёт ближайшее реальное слово ("продолжим") и подставит
- Cmd+Z в течение 5 секунд откатывает замену и **запоминает**: в этом приложении это слово больше не трогать
- Cmd+Shift+X — выделил текст, конвертировал между раскладками одной клавишей

**Что НЕ делает:**

- Не передаёт ввод на серверы (физически не может — нет HTTP-клиента в коде)
- Не показывает рекламу
- Не требует аккаунта или логина
- Не лезет в clipboard, кроме момента нажатия Cmd+Shift+X (и то возвращает оригинальное содержимое)

## Технически

Стек:

- **Go** для ядра — кросс-платформенность, маленький бинарник (4.7 MB), чистый CGo для нативных macOS API
- **CGEventTap** для перехвата клавиатуры (active mode — может подавлять события, что критично для Enter-перехвата)
- **98K русских слов** + Snowball stemmer + suffix stripping для морфологии
- **Levenshtein edit distance 1** для опечаток
- **NSWorkspace notifications** для thread-safe определения активного приложения
- **LaunchAgent** для автозапуска
- 215 unit-тестов, проходят с `-race` (race detector)

## Бизнес-модель: open core

Ядро бесплатное и навсегда под MIT. Платное — то, что увеличивает удобство, но не блокирует базовый use case:

| Free (всегда) | Pro (490 ₽ разово) |
|---|---|
| Автозамена раскладки | Cmd+Shift+X — конвертация выделенного |
| Fuzzy matching | Расширенный fuzzy (edit distance 2) |
| Cmd+Z откат + автообучение | UI настроек вместо config.yaml |
| Tray icon | Дополнительные раскладки (UK, KZ, BY) |
| Автозапуск | Pro-словарь (терминология, сленг) |
| Per-app исключения через config | iCloud sync настроек |

**Почему 490 ₽ разово, а не подписка:**

- Caramba берёт 449 ₽/год = 1 347 ₽ за 3 года = 2 245 ₽ за 5 лет
- Bzz — 490 ₽ один раз. Через 2 года уже сэкономил
- У переключателя раскладки нет серверов, нет облака — runtime-затрат нет, экономики на подписке нет
- Главное возражение в отзывах Caramba: "за что подписка на утилиту"

## Цифры на старте

- Каналы Caramba (Windows): 200K+ установок. На macOS у них кратно меньше — оценочно 5-10K активных платных подписок.
- Целевой рынок: ~50K русскоговорящих Mac-пользователей в РФ + диаспора.
- Конверсия free → paid для open source утилит обычно 3-5%.

**Консервативный сценарий к году:** 10K юзеров → 400 покупок Pro × 490 ₽ ≈ 196K ₽
**Оптимистичный:** 30K юзеров → 1 200 покупок ≈ 588K ₽

Это не "стать миллионером на утилите", это side-project с понятной положительной экономикой и долгосрочным потоком.

## Что я бы сделал по-другому

1. **Сразу под открытое имя.** Я начал с RuSwitch. Прибился к российскому рынку, ограничил фантазию относительно UK/DE/KZ. Через две недели переименовал в Bzz — потерял неделю на rename и пересборку артефактов.

2. **Apple Developer аккаунт с самого начала.** 99 $/год. Без notarization любой пользователь после Gatekeeper'а проходит 8 шагов до первого запуска. С каждым шагом отваливается 30-50%. Подпись — самый дорогой блокер для distribution.

3. **Не тестировать active CGEventTap только в Терминале.** В debug-сборке всё работает идеально. В release-сборке без подписи активный tap молча не активируется на чистой системе. Поймал это поздно.

4. **Документация для юзеров с самого начала.** Скриншоты, GIF-демо, FAQ, troubleshooting. Технари читают код, обычные люди — README. Я слишком долго думал про то, "как круто работает", а не "как поставить".
